# Auto Code Documentation with LangChain & RAG

## About
An automated **code documentation system** built with **LangChain** and **RAG (Retrieval-Augmented Generation)**. It generates context-aware documentation directly from your codebase and related resources, making project maintenance and onboarding easier.

## Features
- Automatic extraction of code context
- Intelligent documentation generation using LLMs
- Retrieval of related resources to enrich documentation
- Supports multiple programming languages (can be extended)
